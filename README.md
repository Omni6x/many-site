# MANY — website

Готовий React/Vite сайт для бренду MANY.

## Як запустити локально

```bash
npm install
npm run dev
```

## Як захостити на Netlify

1. Завантаж цей проєкт на GitHub.
2. В Netlify натисни **Add new site → Import from Git**.
3. Вибери репозиторій.
4. Build command: `npm run build`
5. Publish directory: `dist`
6. Натисни **Deploy**.

## Як швидко перевірити перед деплоєм

```bash
npm install
npm run build
npm run preview
```
